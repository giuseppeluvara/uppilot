# Sample di output (§7)

Cartella dei modelli per la generazione della bozza.

- `template.docx` — se presente, viene usato come **base** per l'export Word
  (intestazione, stili, sezione P.Q.M.). In sua assenza l'export parte da un
  documento vuoto generato a runtime.
- I file `.md`/`.txt` qui dentro possono essere usati come **guida di stile** per
  i prompt di generazione (struttura del parere/sentenza), come da spec §7.

I sample reali verranno forniti dall'utente.
